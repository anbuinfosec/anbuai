- Terminal-style npm command display with $ prefix

- Enhanced hero section with improved npm command styling
- Improved playground UI with gradients and animations
- Updated status page to show detailed system metrics

## [1.0.0] - 2025-11-25

### Added
- Initial release of Anbu AI platform
- Next.js 16 with Turbopack support
- Multilingual support (5 languages: English, Bengali, Japanese, Vietnamese, Hindi)
- AI chat interface with multiple model support
- Image generation capabilities
- Interactive playground
- Comprehensive documentation
- Responsive design for all devices
- Dark mode support
- Model selection interface
- Contact and about pages

### Features
- **Chat API** - Advanced conversational AI
- **Image API** - AI-powered image generation
- **Status API** - Real-time system monitoring
- **Changelog API** - Git-based version history
- **Translation System** - Context-based i18n with localStorage
- **Obfuscation** - Built-in API encryption utility

### Technical
- Next.js 16.0.4 with App Router
- React 19.2.0
- TypeScript 5.9.3
- Tailwind CSS 4.1.17
- Radix UI components
- Zustand for state management
- Lucide icons

### UI Components
- Button, Card, Dialog, Dropdown
- Input, Textarea, Select
- Toast notifications
- Badge, Avatar, Skeleton
- Tabs, Accordion, Collapsible
- And 40+ more components
